Convencion cartesiana usada: do positivo para objeto real frente espejo. f positivo concavo, f negativo convexo. di positivo imagen real (frente), di negativo virtual (detras).

Formulas principales

Ecuacion de espejo 1/f = 1/do + 1/di
Despeje de di di = (f * do) / (do - f)
Despeje de do do = (f * di) / (di - f)
Aumento lateral m = hi/ho = -di/do
Altura de imagen hi = m * ho
Radio y focal R = 2f y f = R/2
Potencia optica P = 1/f(metros)
Casos limite do = f -> di -> infinito, do -> infinito -> di = f
Imagen real/virtual di > 0 real, di < 0 virtual
Orientacion m < 0 invertida, m > 0 derecha
